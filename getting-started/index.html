MyBinder

MyBinder hosts remote JupyterHub servers where the pyiron codebase is already installed. You can even create a new notebook from scratch on the server once it's running - just be aware that MyBinder is meant for testing, so once you close the instance your changes will be lost.

  1. Start the pyiron MyBinder server here. The bootup can take 2-5 minutes as it installs the latest version of pyiron from the repository.

  2. Open the first_steps.ipynb notebook and follow the tutorial at your own pace.

  3. Experiment with your own modifications and new code!

You can also find a number of advanced workflows and specific simulation protocols in the notebooks accompanying papers under publications.
